# Dialog: Add Note

Add-note dialog launched from a contact or partner detail page (mobile bottom sheet variant).

## How to navigate

1. Sign in (see [sign-in](sign-in.md)) on a mobile viewport.
2. Open a contact (or partner) detail page.
3. Tap **Add Note** in the notes panel → opens the bottom sheet (`NCD8q`).
