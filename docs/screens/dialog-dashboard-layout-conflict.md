# Dialog: Dashboard Layout Conflict

Conflict resolution prompt when the local dashboard layout diverges from the server (LWW reconciliation).

## How to navigate

1. Sign in (see [sign-in](sign-in.md)).
2. Land on `/dashboard` and click **Edit dashboard**.
3. While editing, the same user (or a service event) updates the saved layout from another session.
4. Click **Save** locally — the conflict dialog (`MQ4B5`) opens to resolve.
