# Dialog: Confirm Sign Out (Unsaved Work)

Confirmation prompt shown when sign-out is initiated while there is unsaved work in the app.

## How to navigate

1. Sign in (see [sign-in](sign-in.md)).
2. Open a form page (e.g. contact edit) and modify a field so the page is dirty.
3. Open the [profile menu](profile-menu.md) and click **Sign out** → opens the confirmation dialog (`rqHDd`).
