# Profile Menu

User profile menu invoked from the top bar / mobile profile sheet, including the sign-out entry point.

## How to navigate

1. Sign in (see [sign-in](sign-in.md)).
2. From any authenticated page:
   - **Desktop / Profile Menu Open** (`U7ry4`) — click the avatar in the top-right of the app shell.
   - **Mobile / Profile Sheet** (`Dqa28`) — on mobile, tap the avatar in the top bar to open the bottom sheet.

## Designs

Frames in `docs/ui-design.pen`:

| ---------- | -------- | -------- | ----- |
