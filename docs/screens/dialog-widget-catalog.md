# Dialog: Widget Catalog

Catalog browser for adding widgets to the dashboard.

## How to navigate

1. Sign in (see [sign-in](sign-in.md)).
2. Land on `/dashboard`.
3. Click **Edit dashboard** in the page header to enter edit mode.
4. Click **Add widget** → opens the catalog dialog (`O0h0ll` / `Zjl9M` / `C42xUD`).
