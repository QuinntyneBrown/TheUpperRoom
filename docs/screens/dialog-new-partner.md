# Dialog: New Partner

Create-partner dialog (modal on tablet, sheet on mobile). Desktop equivalent is integrated into Partners list flows.

## How to navigate

1. Sign in (see [sign-in](sign-in.md)).
2. Click **Partners** in the side nav.
3. Click **New Partner** in the page header → opens the dialog:
   - **Tablet Modal** (`SmzaX`).
   - **Mobile Sheet** (`kMQVk`).

## Designs

Frames in `docs/ui-design.pen`:

| Frame Name | Frame ID | Viewport | State |
| ---------- | -------- | -------- | ----- |
| Tablet / New Partner Modal | `SmzaX` | Tablet 768×1024 | Modal |
| Mobile / New Partner Sheet | `kMQVk` | Mobile 375×812 | Bottom sheet |
