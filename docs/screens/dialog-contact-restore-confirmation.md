# Dialog: Contact Restore Confirmation

Confirmation prompt before restoring an archived contact.

## How to navigate

1. Sign in (see [sign-in](sign-in.md)).
2. Click **Contacts** → switch to the **Archive** tab (see [contacts-archive](contacts-archive.md)).
3. Click **Restore** on an archived contact row → opens the confirmation dialog (`eO0tH`).
