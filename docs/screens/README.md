# Screens & Dialogs Index

Each markdown in this folder traces a single logical screen or dialog to one or more frames in `docs/ui-design.pen`. Frames listed in each document are the authoritative UI design the implementation must follow.

Open the design file with the Pencil MCP `open_document` tool, then look up frames by ID via `batch_get`.
