# Dialog: Contact Delete Confirmation

Confirmation prompt before soft-deleting a contact. The mobile detailed sheet variant explains restore-from-archive behavior.

## How to navigate

1. Sign in (see [sign-in](sign-in.md)) as a user with the contact-delete role.
2. Click **Contacts** → click any contact row → contact detail page.
3. Click **Delete** in the page header / overflow menu → opens the confirmation:
   - **Modal** (`FZmxK`) on desktop / tablet.
   - **Detailed Sheet** (`d8o9e`) on mobile (with restore-from-archive explanation).
