# Dialog: Associate Contact

Search-and-pick dialog for associating an existing contact to a partner.

## How to navigate

1. Sign in (see [sign-in](sign-in.md)).
2. Click **Partners** → click a partner row → partner detail page.
3. In the **Contacts** panel, click **Associate Contact** → opens the dialog:
   - **Default** (`VhQUS`) — initial render with no query.
   - **Search Results** (`R9TqiO`) — type a query that matches existing contacts.
   - **No Results** (`ADayX`) — type a query that matches nothing.
   - **Mobile Sheet** (`OlgT6`) — bottom sheet variant.
