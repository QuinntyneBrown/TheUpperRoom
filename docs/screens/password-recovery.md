# Password Recovery

Request password reset email.

## How to navigate

No login required.

1. Open the app at `/sign-in`.
2. Click **Forgot password?** → `/password-recovery` (`y1VvT` / `HpwWB`).
3. Enter the account email and submit → **Sent** confirmation (`Zu6hB` / `ph9B9`).
