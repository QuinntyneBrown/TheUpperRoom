# Dialog: Partner Delete Confirmation

Confirmation prompt before deleting a partner.

## How to navigate

1. Sign in (see [sign-in](sign-in.md)) as a user with the partner-delete role.
2. Click **Partners** → click any partner row → partner detail page.
3. Click **Delete** in the page header / overflow menu → opens the confirmation:
   - **Modal** (`Vf7RJ`) on desktop / tablet.
   - **Sheet** (`xye4T`) on mobile.
