# Team

Local team member list with manage actions.

## How to navigate

1. Sign in (see [sign-in](sign-in.md)).
2. Click **Team** in the side nav → `/team` (`Epzpe` / `aPqPR`).
3. To reach the variant:
   - **Manage Sheet** (`TRcte`) — on mobile, tap the overflow / manage action on a team member row.

## Designs

Frames in `docs/ui-design.pen`:

| Frame Name | Frame ID | Viewport | State |
| ---------- | -------- | -------- | ----- |
| Desktop / Team | `Epzpe` | Desktop 1440×1024 | Default |
| Mobile / Team | `aPqPR` | Mobile 375×812 | Default |
| Mobile / Team - Manage Sheet | `TRcte` | Mobile | Per-member manage bottom sheet |
