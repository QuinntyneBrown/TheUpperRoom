# Dialog: Delete Note Confirmation

Confirmation prompt before deleting a note from a contact (or partner) detail page.

## How to navigate

1. Sign in (see [sign-in](sign-in.md)).
2. Open a contact (or partner) detail page that has at least one note.
3. On a note row, click the **Delete** action → opens the confirmation dialog (`CBNXl`).
