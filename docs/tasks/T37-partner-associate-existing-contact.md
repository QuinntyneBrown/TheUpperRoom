# T37 — Associate existing contact with partner

**Phase**: 5 — Partner and hackathon management
**Area**: Partner, Contacts
**Requirements**: L1-004, L2-018
**Source**: Screen-Level Missing Inventory — "Associate existing contact with partner"

## Goal

Provide a picker on partner detail that lets the user attach an existing contact to the partner.

## Scope

- Trigger: "Add" or "+ Associate contact" button on the partner contacts panel.
- Picker dialog with searchable contact list, recent contacts, and selection state.
- Loading, empty, and error states for the picker.
- Confirmation toast on association.

## Acceptance criteria

- [ ] Picker dialog designed with all states.
- [ ] Search highlights match (links to T30).
- [ ] Mobile bottom-sheet variant per T21.
