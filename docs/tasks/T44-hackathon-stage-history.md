# T44 — Hackathon stage history

**Status**: Accepted
**Phase**: 5 — Partner and hackathon management
**Area**: Hackathon
**Requirements**: L1-005, L2-023, L2-025
**Source**: Screen-Level Missing Inventory — "Hackathon stage history"

## Goal

Show a chronological audit of stage transitions for a hackathon.

## Scope

- Timeline component on hackathon detail.
- Each entry: from-stage → to-stage, actor, timestamp, optional note.
- Empty state when no transitions yet.
- Live append on stage change.

## Acceptance criteria

- [ ] Timeline designed with sample entries and empty state.
- [ ] Mobile variant collapsible per T42.
