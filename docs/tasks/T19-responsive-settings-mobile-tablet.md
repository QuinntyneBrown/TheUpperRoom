# T19 — Mobile and tablet settings

**Phase**: 3 — Responsive coverage and clipping fixes
**Area**: Settings, Responsive
**Requirements**: L1-009, L2-039, L2-040
**Source**: Screen-Level Missing Inventory — "Mobile/tablet ... settings"

## Goal

Provide responsive variants of the settings screen.

## Scope

- `Mobile / Settings`: stacked sections, in-page anchors or sectioned navigation.
- `Tablet / Settings`: two-column where helpful.
- Forms within settings reflow to a single column on mobile.

## Acceptance criteria

- [ ] Mobile and tablet settings screens exist.
- [ ] Section headers are sticky or otherwise easily reachable.
- [ ] All inputs and toggles meet hit-target requirements.
