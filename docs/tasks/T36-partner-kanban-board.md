# T36 — Partner Kanban board

**Phase**: 5 — Partner and hackathon management
**Area**: Partner
**Requirements**: L1-004, L2-021
**Source**: Screen-Level Missing Inventory — "Partner Kanban board"

## Goal

Add a desktop Kanban view of partners grouped by funnel stage, with drag-and-drop stage transitions.

## Scope

- `Desktop / Partners - Board`: three columns (Lead, In Funnel, Confirmed Partner) with partner cards.
- Drag-in-progress state, drop-target highlight, snap-back on cancel.
- Card content: name, city, owner, last activity.
- Multi-select filter integration (city, owner, etc.).
- Empty-stage state per column.

## Acceptance criteria

- [ ] Board view exists at desktop width with all three columns and several sample cards.
- [ ] Drag/drop interaction states represented.
- [ ] Toggle between list and board views in the partner page header.
