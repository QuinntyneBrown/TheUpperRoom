# T42 — Hackathon detail screen

**Phase**: 5 — Partner and hackathon management
**Area**: Hackathon
**Requirements**: L1-005, L2-025
**Source**: Screen-Level Missing Inventory — "Hackathon detail screen"

## Goal

Add a dedicated detail screen for a single hackathon, beyond the current overview.

## Scope

- Header: name, dates, host city, current 4 D stage, edit button, archive button.
- Sections: sponsoring partners, teams/products (T45), stage history (T44), notes.
- Order: products first, then partners, then history.
- Mobile collapsible sections.
